# Marina Imports

Landing page pronta para GitHub Pages.

## Publicação
1. Envie todo o conteúdo desta pasta para a raiz do repositório.
2. No GitHub: Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /(root).
6. Save.

As imagens, o CSS e o JavaScript estão separados em `assets/`.
